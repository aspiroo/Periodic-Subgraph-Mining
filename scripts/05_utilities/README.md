# Stage 5: Utilities

Utility scripts for specialized analyses.

## Overview

This stage contains utility tools for:
- Calculating pattern purity scores
- Counting subgraph occurrences
- Extracting GO terms from enrichment results

## Scripts

- **purity.py** - Calculate purity scores for gene patterns
- **subgraph_count.py** - Count and analyze subgraph statistics
- **extract_go_terms.py** - Extract GO terms from FlyEnrichr results

## Usage

These scripts are typically run after main analysis to provide additional insights and quality metrics.

## Input

Varies by script - see individual script documentation

## Output

Analysis reports and statistics in `results/utilities/`
